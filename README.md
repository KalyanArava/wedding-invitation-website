# 💍 Wedding Invitation Website

A beautiful and interactive wedding invitation website built using HTML, Tailwind CSS, and GSAP animations.

---

## ✨ Features

- 🎬 Cinematic curtain opening animation
- 💖 Elegant couple section with images
- 📅 Wedding event timeline
- 📍 Google Maps integration (Get Directions)
- 🎵 Background music control
- 📱 Fully responsive design
- ✨ Smooth animations using GSAP

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- GSAP (GreenSock Animation Platform)
